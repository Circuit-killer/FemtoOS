\subsection{Detailed Description}
We use two different types of tick, \hyperlink{union_ttick}{Ttick} and \hyperlink{union_textended_tick}{Textended\-Tick}. The former is used if we have standard tick interrupts (i.\-e. equidistant interrupts) and there is no further need for extra storage. The 16 bit tick counter is kept in two separate bytes, which is more efficient as one 16 bit variable for direct access. However, sometimes it is convenient to have the value as 16 bit word at hand, thus we define a union.
